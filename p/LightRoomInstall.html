How to Install LightRoom Free

Firstly Download the setup from below link Then unzip the File

If you can't unzip the file then Download WinRar

right click on the setup And choose Run as Administrator As the installation Then click on next click on browser button Click on This pc (or "My Computer" on Window 8 or below) select "Local Disk (C)" Then click on "Install" unchecked the "Run Adobe Photoshop Lightroom" check box then click on "Finish" Open file location right click on the LightroomPortable.exe file And choose Run as Administrator click on "Create a New Catalog..." button Go to the previous location and type "adobe photoshop Lightroom 5 pro" (as your choice) in File name section and click on "Save" and the Installation has been done
